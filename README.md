---

## Demo

Open `index.html` in any modern browser to use the app.

---

## Features

- Add, view and delete expenses (amount, date, category, description)
- Manage custom categories with color selection
- Dashboard with:
  - Total spending
  - This month's spending
  - Daily average spending
- Charts powered by Chart.js:
  - Spending by category (pie)
  - Daily spending over the last 30 days (line)
  - Monthly totals for the last 12 months (bar)
- Settings panel with reminder frequency (UI prototype)
- Data persistence using `localStorage`
- Mobile-friendly layout and a bottom navigation bar for small screens
- "Danger Zone" option to clear all local data
